# Feeds

The site has a main [Blog](/blog) feed that's served in different flavors:

- [RSS feed](/rss.xml)
- [Atom feed](/atom.xml)
- [JSON feed](/feed.json)

Want to learn more about RSS? Check out [About Feeds](https://aboutfeeds.com/) which has a good explanation on what is RSS and how to use it.
